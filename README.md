# 前庭電気刺激(GVS)実験用

前庭電気刺激(GVS)の実験用リポジトリ．

耳の後ろ辺りに電極パッドを貼り付けて電流を流すと加速度を感じます．

PC -(SPI)-> AVR -(SPI)-> DAC -> OPアンプ(LMC662CN) -> MOSFET -> DC昇圧回路 -> 電極パッド


## ハードウェア

```
                                                       5Vpp              9Vpp                           200V
+------+               +----------+          +-------+      +----------+      +--------+    +-----------+    +------------+    ************
|  PC  |  -(USB-SPI)-> | AVR      | -(SPI)-> |  DAC  | ---> | OPアンプ | ---> | MOSFET | -> | DC-DC昇圧 | -> | 電極パッド | -> * 前庭器官 *
|      |               | ATMEGA88 |          |       |      |          |      |        |    | 9V->200V  | -> |            |    *          *
+------+               +----------+          +-------+      +----------+      +--------+    +-----------+    +------------+    ************

```
